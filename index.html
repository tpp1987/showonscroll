What's this all about?

This page is a test. I made it because I'm learning JavaScript, and I wanted to try something out.

Here, you will find a selection of random images from Unsplash.com, laid out in a single column.

The main purpose of this project is to display a set of social icons at the right of the page when you scroll beyond a certain point. The icons are not actually links.

I also wanted to learn how to deploy to Netlify. That's it.
